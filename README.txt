El proyecto consiste en el desarrollo de un sitio web para la empresa “Mente Libre Ecoturismo Accesible”, cuyo objetivo es ofrecer experiencias turísticas inclusivas para personas con discapacidad y grupos vulnerables. A lo largo del proyecto, se trabajó en distintas etapas que incluyen la recolección de requerimientos mediante una reunión con el emprendedor, el diseño inicial del mockup en HTML, la mejora visual utilizando CSS, y finalmente, la migración del contenido a una plantilla profesional llamada “Architecture”.

El sitio web incorpora secciones como misión, visión, galería, información de servicios, redes sociales y contacto, con un fuerte enfoque en la accesibilidad digital (lectores de pantalla, contraste adecuado y navegación inclusiva). El trabajo fue realizado en equipo, permitiendo aplicar conocimientos prácticos de diseño web, CSS, y JavaScript básico, con énfasis en la compatibilidad multiplataforma y la experiencia del usuario.
